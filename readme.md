## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  pip install fastapi uvicorn sqlalchemy "python-jose[cryptography]" "passlib[bcrypt]"
```

Start the server

```bash
  uvicorn main:app --reload
```
